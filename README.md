## use-Popcorn Project

Welcome to the use-Popcorn Project from Jonas Schmedtmann's "The Ultimate React Course 2023: React, Redux & More."

### Getting Started

To get started with this project, follow these steps:

1. **Clone**: Clone this repository to your local machine.
2. **Install**: Install project dependencies using `npm install`.
3. **Run**: Start the development server with `npm start`.

- React
- HTML
- CSS
